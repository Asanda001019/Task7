 <h1> Task7 Todo list with register and login</h1>

 
## Run Locally
Clone the project
```bash
  https://github.com/Asanda001019/Task7.git
```
Go to the project directory
```bash
  cd Task7
```
Install dependencies
```bash
  npm install
```
Start the react server
```bash
  npm run dev
nodemon server.js
```
## Tech Stack
**Client:** React JS, HTML5, JavaScript, CSS3
